Panduan Instalasi & Uninstall

Install

Buka PowerShell, lalu jalankan perintah berikut:

irm https://sambungkata.sakum.my.id/app/installer/install.ps1 | iex

Uninstall

Untuk menghapus game, jalankan perintah ini di PowerShell:

irm https://sambungkata.sakum.my.id/app/uninstall/uninstall.ps1 | iex

Install

Buka terminal, lalu jalankan perintah berikut:

curl -sSL https://sambungkata.sakum.my.id/app/installer/install.sh | bash

Uninstall

Untuk menghapus game, jalankan perintah ini di terminal:

curl -sSL https://sambungkata.sakum.my.id/app/uninstall/uninstall.sh | bash

Install APK

Buka file APK yang sudah didownload. Jika muncul peringatan, aktifkan izin "Install dari Sumber Tidak Dikenal" di pengaturan HP kamu.

Uninstall

Tekan lama pada ikon aplikasi Sambung Kata di menu HP kamu, lalu pilih Uninstall.
