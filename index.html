Cara Instalasi

1

Windows

Download file install.bat, lalu klik kanan dan pilih "Run as Administrator" jika diperlukan untuk membuat shortcut.

2

Linux

Buka terminal di folder download, lalu jalankan perintah berikut:

chmod +x install.sh
./install.sh

*Gunakan sudo jika instalasi membutuhkan akses sistem.

3

Android

Buka file APK yang sudah didownload. Jika muncul peringatan, aktifkan izin "Install dari Sumber Tidak Dikenal" di pengaturan HP kamu.
